## 1. 로그인

학생은 이메일/비밀번호가 아닌 **학원코드 + 학생코드 + PIN** 방식으로 로그인합니다.

### 접속 경로
```
http://localhost:3000/login
```

### 로그인 흐름
1. 로그인 페이지에서 **"학생"** 버튼 선택
2. 다음 3가지 정보 입력:
   - **학원코드**: 학원에서 안내받은 코드 (예: `DEMO-1234`)
   - **학생코드**: 나의 학생 코드 (예: `2025-001`)
   - **PIN**: 4자리 숫자 PIN
3. **"로그인"** 버튼 클릭
4. 로그인 성공 시 학생 홈 화면으로 이동

### PIN을 잊어버린 경우
- 강사 또는 원장 선생님께 PIN 초기화를 요청하세요.
- 초기화 후 새 PIN을 부여받습니다.

## 2. 홈 화면

**경로:** `/student/home`

로그인 후 첫 번째로 보이는 화면입니다.

### 홈 화면에서 볼 수 있는 것들
- **다음 수업 일정**: 가장 가까운 수업 날짜, 시간, 과목
- **제출 마감 임박한 과제**: 마감일이 다가온 과제 목록
- **새 복습 자료**: 선생님이 새로 올린 복습 자료
- **미확인 Q&A 답변**: 내 질문에 달린 새 답변

## 3. 출석 현황 확인

**경로:** `/student/attendance`

내 출석 기록을 확인할 수 있습니다.

### 확인 가능한 내용
- **월별 출석 현황**: 이번 달 출석 달력
- **출석 통계**:
  - 출석 횟수
  - 결석 횟수
  - 지각 횟수
  - 출석률 (%)
- **출석 상태 색상 구분**:
  - 초록색: 출석
  - 노란색: 지각 또는 조퇴
  - 빨간색: 결석
  - 회색: 수업 없음

> 출석 기록은 선생님이 직접 입력합니다. 본인 기록이 잘못된 경우 선생님께 말씀해 주세요.

## 4. 과제 제출

**경로:** `/student/assignments`

선생님이 낸 과제를 확인하고 제출합니다.

### 과제 목록 확인
- 현재 진행 중인 과제 목록
- 각 과제의 마감일, 제출 상태 표시
- 상태 구분:
  - **미제출**: 아직 제출하지 않음 (빨간 표시)
  - **제출 완료**: 제출한 과제
  - **첨삭 완료**: 선생님이 첨삭해준 과제

### 과제 제출 흐름
1. 과제 목록에서 제출할 과제 클릭
2. 과제 상세 페이지(`/student/assignments/[id]`) 확인
3. 과제 내용과 지시사항 읽기
4. **"제출하기"** 버튼 클릭
5. 제출 방식에 따라:
   - **문제집 유형**: 완료 여부 체크
   - **에세이 유형**: 텍스트 직접 입력
   - **이미지 유형**: 사진 파일 업로드
6. **"제출"** 버튼 클릭

### 제출물 확인 및 첨삭 보기
1. 과제 목록에서 제출 완료된 과제 클릭
2. 내가 제출한 내용 확인
3. 선생님이 첨삭을 완료한 경우 → **첨삭 내용** 확인 가능
4. 첨삭 피드백을 바탕으로 복습

## 5. 복습 자료 확인

**경로:** `/student/review`

선생님이 수업 후 생성한 AI 복습 자료를 확인합니다.

### 복습 자료 목록
- 수업별로 정리된 복습 자료 목록
- 최신순으로 표시

### 복습 자료 상세 확인
**경로:** `/student/review/[id]`

복습 자료에 포함된 내용:
- **핵심 개념 정리**: 수업에서 다룬 중요한 내용 요약
- **복습 퀴즈**: 이해도 확인용 문제
- **다음 수업 준비사항**: 다음 시간 전 준비해야 할 것

### 복습 활용 방법
```
수업 후 1-2일 내에 복습 자료 확인
→ 핵심 개념 다시 읽기
→ 퀴즈 풀기
→ 틀린 부분 교재에서 다시 찾아보기
→ 다음 수업 준비사항 미리 챙기기
```

## 6. Q&A 질문하기

**경로:** `/student/qna`

수업 중 또는 수업 후 궁금한 점을 질문합니다.

### 질문 등록 흐름
1. Q&A 메뉴 접속
2. **"질문하기"** 버튼 클릭
3. 질문 내용 입력
4. **"등록"** 버튼 클릭

### 답변 확인
- 질문 등록 후 목록에서 내 질문 확인
- 답변 상태:
  - **답변 대기**: 아직 답변이 없음
  - **AI 답변**: AI가 자동으로 답변 제공
  - **선생님 답변**: 선생님이 직접 답변 완료
- 답변이 달리면 홈 화면에서 알림 표시

> Q&A는 수업 관련 학습 질문에 활용하세요. 출석, 결제 관련 문의는 직접 학원에 연락해 주세요.

## 전체 학생 사용 흐름

```
매 수업일
└── 로그인
    └── 홈에서 오늘 수업 일정 확인

수업 후
└── 복습 자료 확인
    ├── 핵심 개념 정리 읽기
    └── 복습 퀴즈 풀기

과제 마감 전
└── 과제 목록 확인
    ├── 과제 내용 확인
    ├── 제출하기
    └── 제출 후 선생님 첨삭 확인

궁금한 점 생기면
└── Q&A에 질문 등록
    └── 답변 확인
```

## 자주 묻는 질문

**Q. 로그인이 안 돼요.**
A. 학원코드, 학생코드, PIN을 정확히 입력했는지 확인하세요. 계속 안 된다면 선생님 또는 원장 선생님께 문의하세요.

**Q. PIN을 잊어버렸어요.**
A. 담당 선생님 또는 원장 선생님께 PIN 초기화를 요청하세요.

**Q. 제출한 과제를 수정하고 싶어요.**
A. 마감일 전에는 다시 제출이 가능합니다. 마감 후에는 선생님께 직접 문의하세요.

**Q. 복습 자료가 안 보여요.**
A. 선생님이 복습 자료를 생성하지 않았거나 아직 공개하지 않은 것입니다. 선생님께 문의하세요.

**Q. 출석이 잘못 기록되어 있어요.**
A. 담당 선생님께 직접 말씀해 주세요. 선생님이 수정할 수 있습니다.

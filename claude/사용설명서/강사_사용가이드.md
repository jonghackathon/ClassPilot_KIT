## 1. 로그인

### 접속 경로
```
http://localhost:3000/login
```

### 로그인 흐름

## 2. 출석 관리

**경로:** `/teacher/attendance`

수업 출석을 실시간으로 기록합니다.

### 출석 기록 흐름
1. 출석 관리 페이지 접속
2. 날짜와 수업(반) 선택
3. 학생 목록에서 각 학생의 출석 상태 선택:
   - **출석**: 정상 참석
   - **지각**: 늦게 도착
   - **조퇴**: 일찍 귀가
   - **결석**: 미참석
4. **"저장"** 버튼으로 일괄 저장

### 일괄 출석 처리
- 전체 학생 선택 후 상태 일괄 변경
- 특정 학생만 개별 변경 가능

### 출석 통계 확인
- 학생별 월간 출석률 자동 계산
- 연속 결석 학생 강조 표시
- 출석 통계는 이탈예측에 자동 반영

## 3. 과제 관리

**경로:** `/teacher/assignments`

### 과제 생성 흐름
1. 과제 관리 페이지에서 **"과제 추가"** 버튼 클릭
2. 아래 정보 입력:
   - 과제 제목
   - 과제 유형: **문제집** / **에세이** / **이미지**
   - 대상 반 선택
   - 마감일 설정
   - 과제 파일 첨부 (선택)
   - 과제 설명/지시사항 입력
3. **"생성"** 버튼 클릭 → 해당 반 전체 학생에게 자동 배포

### 일괄 과제 생성
- 여러 반에 동시에 동일한 과제 배포 가능

### 제출물 확인 흐름
1. 과제 목록에서 과제 클릭
2. 과제 상세 페이지(`/teacher/assignments/[id]`)에서 제출 현황 확인:
   - 제출 완료 / 미제출 학생 목록
   - 제출 일시 및 내용 확인
3. 제출물 클릭 → 첨삭 작성

### 첨삭 작성 흐름
1. 제출물 상세에서 학생 제출 내용 확인
2. 첨삭 내용 입력 (텍스트)
3. **에세이 유형**: AI 첨삭 초안 생성 가능 (Claude AI 활용)
4. **"첨삭 저장"** 버튼 클릭 → 학생 화면에 즉시 반영

### 과제 유형별 특이사항
| 유형 | 설명 | AI 지원 |
|------|------|---------|
| 문제집 | 교재 문제 과제 | - |
| 에세이 | 글쓰기 과제 | AI 첨삭 초안 생성 |
| 이미지 | 손글씨/사진 제출 | - |

## 4. 학생 진도 관리

**경로:** `/teacher/progress`

담당 반 학생들의 전체 진도를 한눈에 확인합니다.

### 확인 가능한 정보
- 학생별 커리큘럼 진도율
- 과제 제출률
- 최근 수업 출석 현황
- 학생별 강사 메모 요약

### 활용 방법
- 진도가 느린 학생 식별 → 개별 보충 수업 계획
- 과제 미제출이 잦은 학생 → 학부모 연락 요청 (관리자에게 전달)

## 5. AI 코파일럿

**경로:** `/teacher/copilot`

수업 중 학생 질문에 실시간 AI 답변을 제공하는 기능입니다.

### 코파일럿 세션 시작 흐름
1. 코파일럿 메뉴 접속
2. **"새 세션 시작"** 버튼 클릭
3. 수업(Lesson) 선택
4. 세션 상태가 **"활성"** 으로 변경

### 수업 중 활용 흐름
```
학생이 질문
→ 강사가 코파일럿에 질문 입력
→ AI가 즉시 답변 생성 (Claude / OpenAI 활용)
→ 강사가 답변 확인 후 학생에게 전달
→ 질문/답변이 자동 저장
```

### 세션 종료
- 수업 종료 후 **"세션 완료"** 버튼 클릭
- 세션 내 모든 질문/답변이 저장되어 복습 자료 생성에 활용

### 세션별 질문 목록 확인
**경로:** `/teacher/copilot/[lessonId]`
- 해당 수업에서 다뤄진 모든 질문 목록
- 각 질문에 대한 AI 답변 확인

## 6. 수업 녹음 및 요약

**경로:** `/teacher/recording`

수업 내용을 녹음하고 AI가 자동으로 요약합니다.

### 녹음 시작 흐름
1. 녹음 메뉴 접속
2. 수업 선택
3. **"녹음 시작"** 버튼 클릭 → 마이크 권한 허용
4. 수업 진행
5. **"녹음 종료"** 버튼 클릭

### AI 요약 생성
- 녹음 종료 후 AI가 자동으로 요약 생성 (처리 시간 소요)
- 요약 내용: 주요 개념, 다뤄진 문제, 핵심 포인트
- 요약 완료 후 알림 표시

### 녹음 상세 확인
**경로:** `/teacher/recording/[id]`
- 녹음 파일 재생
- AI 요약 내용 확인 및 수정
- 복습 자료 생성으로 연동 가능

### 녹음 처리 상태
| 상태 | 의미 |
|------|------|
| PROCESSING | AI 처리 중 |
| COMPLETED | 요약 완료 |
| FAILED | 처리 실패 (재시도 필요) |

## 7. 복습 자료 생성

**경로:** `/teacher/` (녹음 상세에서 연동)

수업 내용을 바탕으로 학생용 복습 자료를 AI가 자동 생성합니다.

### 복습 자료 생성 흐름
1. 녹음 상세 페이지에서 **"복습 자료 생성"** 버튼 클릭
2. AI가 녹음 요약을 바탕으로 복습 자료 생성:
   - 핵심 개념 정리
   - 복습 퀴즈 문제
   - 다음 수업 준비사항
3. 생성된 자료 확인 및 수정
4. **"학생에게 공개"** 설정 → 학생 화면에서 확인 가능

## 8. 월간 보고서

학생별 월간 성과를 정리한 보고서를 생성합니다.

### 보고서 생성 흐름
1. 학생 진도 또는 관리자 지정으로 보고서 요청
2. **"월간 보고서 생성"** 버튼 클릭
3. AI가 해당 월의 데이터를 분석하여 초안 작성:
   - 출석 현황
   - 과제 제출 현황
   - 학습 진도
   - 강사 의견 (수동 입력)
4. 내용 검토 및 수정
5. 저장 (학원관리자도 열람 가능)

## 9. 메모 관리

학생별 메모를 작성하여 관리합니다.

### 메모 카테고리
| 카테고리 | 용도 |
|----------|------|
| NOTICE | 공지 사항 |
| NOTABLE | 특이사항 |
| STUDENT_NOTE | 학생 관찰 노트 |
| OTHER | 기타 |

### 메모 작성 흐름
1. 학생 상세 또는 메모 메뉴 접속
2. **"메모 추가"** 버튼 클릭
3. 카테고리 선택 및 내용 입력
4. 저장

## 10. 주간 수업 노트

수업 단위로 수업 내용, 특이사항, 다음 수업 준비사항을 기록합니다.

### 주간 노트 작성 흐름
1. 수업 완료 후 주간 노트 메뉴 접속
2. 해당 수업 선택
3. 아래 내용 입력:
   - 이번 수업에서 다룬 내용
   - 학생 반응 및 특이사항
   - 다음 수업 준비사항
4. 저장

## 전체 강사 업무 흐름

```
수업 전
└── 코파일럿 세션 시작
    └── 수업 자료 및 과제 확인

수업 중
└── 출석 기록 (수업 시작 시)
    ├── 코파일럿으로 학생 질문 AI 답변 활용
    └── 수업 녹음 (선택)

수업 후
└── 코파일럿 세션 종료
    ├── 복습 자료 생성 (녹음 요약 → 복습 자료)
    ├── 주간 수업 노트 작성
    └── 과제 확인 및 첨삭

매주
└── 학생 진도 현황 확인
    └── 특이 학생 메모 업데이트

매월
└── 월간 보고서 생성
    └── 내용 검토 후 관리자에게 공유
```

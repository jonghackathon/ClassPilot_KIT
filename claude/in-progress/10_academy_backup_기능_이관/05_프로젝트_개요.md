# 프로젝트 개요 — ClassPilot

**작성일:** 2026.04.10
**버전:** v2.0 (academy_backup 기능 이관 반영)

---

## 1. 한 줄 정의

> 초·중·고등 독서논술 학원을 위한 AI 통합 관리 플랫폼.
> 출결·과제·커리큘럼·수업 진도를 하나로 연결하고, AI가 에세이 첨삭과 학부모 보고서를 자동으로 만들어주는 시스템.

---

## 2. 프로젝트 배경

### 2.1 실제 현장에서 출발

이 프로젝트는 **실제 운영 중인 독서논술 학원**의 관리 시스템(academy_backup)을 기반으로 합니다.
강사가 매일 사용하면서 검증된 기능을 가져오고, 여기에 다중 역할(운영자/강사/수강생)과 새로운 AI 기능을 추가합니다.

### 2.2 해결하려는 문제

| 대상 | 문제 | 현재 상황 |
|------|------|----------|
| **강사** | 첨삭 시간 과다 | 학생당 에세이 첨삭 30분+. 학생 수 × 주 2회 = 주당 10시간+ |
| **강사** | 수업 기록 없음 | 어디까지 했는지, 다음 시간 뭘 할지 기억에 의존 |
| **강사** | 학부모 보고서 | 매월 학생별 보고서 수작업. 학생 30명 × 1시간 = 30시간/월 |
| **강사** | 반복 업무 | 출결+숙제 체크+과제 생성+진도 기록이 흩어져 있음 |
| **운영자** | 이탈 감지 못함 | 학생이 그만둔 후에야 인지 |
| **운영자** | 수납 추적 어려움 | 미납 학생 수기 확인 |
| **수강생** | 복습 부재 | 수업 끝나면 뭘 복습해야 할지 모름 |
| **학부모** | 정보 부족 | 아이가 뭘 배우는지, 어떻게 성장하는지 알 수 없음 |

### 2.3 핵심 가치

```
1. 강사 반복 업무 자동화
   출결+숙제+과제+진도를 한 흐름으로 → 수업에 집중

2. AI 첨삭으로 피드백 품질+속도 동시 향상
   사진 찍으면 AI가 첨삭 → 강사가 수정 → 저장

3. 학부모 보고서 자동 생성
   출결+과제+진도 데이터 → AI 성장 평가 → 인쇄/전달

4. 이탈 조기 감지
   출결+과제+접속 패턴으로 위험 학생 자동 감지
```

## 3. 대상 학원

### 3.1 학원 유형

**독서논술 전문 학원** (국어/논술/에세이 중심)

- 초등 저학년 (1~3학년): 독서 습관 형성, 기초 글쓰기
- 초등 고학년 (4~6학년): 독서논술, 에세이 기법, 사고력 훈련
- 중등 (1~3학년): 서평, 비평문, 논술 심화
- 고등 (1~3학년): 대입 논술, 수시 자소서, 심화 비평

### 3.2 학년 체계

```
초등부:
  초등 1학년 · 초등 2학년 · 초등 3학년
  초등 4학년 · 초등 5학년 · 초등 6학년

중등부:
  중등 1학년 · 중등 2학년 · 중등 3학년

고등부:
  고등 1학년 · 고등 2학년 · 고등 3학년
```

### 3.3 수업 구조

```
반 구성 예시:
  초4 - 친절한 설명 Logos         (독서논술 기초, 사고 기법)
  초5 - 고전1000                 (문화·역사·과학·예술 독서)
  초6 - 고전1300                 (동양·서양 고전 독서)
  초5~6 - Monad 추론의 세계 1100  (추론·비판·창의 사고)
  중1 - 렛잇비 EGO 서평           (텍스트 분석, 서평 작성)
  중2~3 - 심화 논술               (논증 구조, 비평문)
  고1~2 - 대입 논술 기초          (주제 분석, 논술 구조)
  고3 - 수시 논술 실전            (실전 연습, 첨삭 집중)

수업 방식:
  주 1~2회, 회당 90~120분
  수업 = 독서 → 토론 → 글쓰기 → 첨삭
  과제 = 에세이(주 과제) + 독서과제(보조) + 어휘(기초)
```

## 4. 사용자 역할

### 4.1 강사 (핵심 사용자)

실제로 매일 사용하는 주 사용자. 모든 기능의 중심.

**일과 흐름:**
```
수업 전  → 출결 입력 + 숙제 확인 (O/X + 사유)
수업 중  → AI 코파일럿으로 수업 보조 / 수업 내용 기록
수업 후  → 과제 생성 (자동 or 수동) / 녹음 요약
첨삭 시  → 학생 글 사진 촬영 → AI 첨삭 → 수정 → 저장
월말    → 보고서 생성 → AI 성장 평가 → 인쇄 → 학부모 전달
수시    → 메모 기록 (학생 특이사항, 학부모 통화 등)
```

**사용하는 기능:**
- 출결 관리 (숙제 체크 포함)
- 과제 관리 (생성 + 이미지 업로드 + AI 첨삭)
- 수업 진도 (기록 + 과제 자동 생성)
- 커리큘럼 (차시 참조)
- 월간 보고서 (AI 성장 평가 + 인쇄)
- AI 코파일럿 (수업 중 보조)
- 녹음 요약 (수업 후 정리)
- 질문봇 관리 (학생 질문 관리)
- 메모 (특이사항 기록)
- 이탈 현황 (담당 학생)

### 4.2 운영자 (관리자)

학원 전체 운영을 관리. 주로 데스크탑에서 사용.

**사용하는 기능:**
- 대시보드 (전체 현황 KPI)
- 학생 관리 (등록/수정/상세)
- 강사 관리
- 반 관리
- 커리큘럼 관리 (생성/수정)
- 시간표 (학생 배정 포함)
- 수강료 관리
- 이탈 예측 레이더
- 민원 관리

### 4.3 수강생

스마트폰에서 간단하게 확인. 최소한의 인터랙션.

**사용하는 기능:**
- 홈 (오늘 수업, 과제, 복습)
- 내 출결 (조회만)
- 내 과제 (작성 + 제출)
- 복습 자료 (AI 요약 + 퀴즈)
- 질문하기

## 5. 핵심 기능 요약

### 5.1 기본 학원 관리 (LMS)

| 기능 | 설명 |
|------|------|
| 출결 관리 | 출석/지각/조퇴/결석 + 숙제 O/X + 결석 사유 + 3뷰 모드 |
| 과제 관리 | 3유형(에세이/독서/어휘) + 이미지 업로드 + 일괄 생성 + 커리큘럼 연동 |
| 커리큘럼 | 클래스→단계→차시 계층 + 프리로드 데이터 |
| 수업 진도 | 날짜별 기록 + 과제 자동 생성 + 캘린더 뷰 |
| 시간표 | 주간 그리드 + 학생 배정 + 색상 코딩 |
| 학생 관리 | 초1~고3 + 반 배정 + 메모 + 이탈 위험도 |
| 수강료 | 납부 추적 (납부/미납/부분) |
| 메모 | 4카테고리 + 아카이브 + 검색 |

### 5.2 AI 기능

| 기능 | AI 모델 | 설명 |
|------|---------|------|
| **에세이 첨삭** | Claude Opus (멀티모달) | 학생 글 사진/텍스트 → 잘한점+교정+피드백+발전방향 |
| **월간 성장 평가** | Claude Opus | 출결+과제+진도 → 성장/강점/방향 보고서 |
| **수업 코파일럿** | Claude Sonnet (SSE) | 수업 중 질문 → 4카드(초보자/코드/심화/판서) |
| **녹음 요약** | Whisper + Claude | 수업 녹음 → 전사 → 요약 |
| **Q&A 봇** | Claude Sonnet | 학생 질문 → AI 1차 응답 → 강사 보완 |
| **이탈 예측** | 규칙 + Claude | 출결+과제+접속 패턴 → 위험도 점수 |
| **민원 초안** | Claude Sonnet | 학부모 민원 → 응답 초안 자동 생성 |
| **복습 요약** | Claude Sonnet | 수업 후 → 3줄 요약 + 퀴즈 3문제 |

### 5.3 실사용에서 검증된 기능 (academy_backup 출처)

| 기능 | 검증 근거 |
|------|---------|
| 출결+숙제 동시 체크 | 강사가 수업 시작 시 한 화면에서 처리 |
| 이미지 기반 AI 첨삭 | 손글씨 사진 → Claude 멀티모달 → 구조화된 피드백 |
| 수업 기록 → 과제 자동 생성 | "오늘 다룬 내용" 입력 + 체크 → 반 전체 과제 생성 |
| 월간 보고서 AI 성장 평가 | 학부모에게 매월 전달, 재원율에 직접 영향 |
| 커리큘럼 계층 구조 | 독서논술 특화 (차시별 교재+테마 관리) |
| 메모 시스템 | 학생 특이사항, 학부모 통화 기록 등 즉시 기록 |

## 8. 차별화 포인트

### 기존 학원 관리 솔루션과의 차이

| 항목 | 기존 솔루션 | ClassPilot |
|------|-----------|-----------|
| 중심 기능 | 출결 + 수납 | 출결 + 과제 + 진도 + AI |
| AI 활용 | 없음 | 첨삭, 보고서, 코파일럿, 이탈예측 |
| 과제 관리 | 제목만 기록 | 이미지 업로드 + AI 첨삭 + 프로세스 추적 |
| 학부모 소통 | 수동 문자/전화 | AI 성장 보고서 자동 생성 + 인쇄 |
| 커리큘럼 | 없음 | 계층 구조 (클래스→단계→차시) |
| 수업 기록 | 없음 | 날짜별 기록 + 과제 자동 생성 |
| 이탈 관리 | 없음 | 다중 요인 자동 감지 + 상담 워크플로우 |
| 수강생 포털 | 없음 | 출결/과제/복습/질문 |

### 실사용 검증 기반

```
academy_backup에서 강사가 실제 사용하며 검증된 기능:
  ✅ 출결+숙제 동시 체크 → 한 화면, 한 번에
  ✅ 이미지 → AI 첨삭 → 수정 → 저장 플로우
  ✅ 수업 기록 → 과제 자동 생성 (1클릭)
  ✅ 월간 보고서 → AI 성장 평가 → 인쇄
  ✅ 커리큘럼 차시 관리 (독서논술 특화)
  ✅ 메모 시스템 (수시 기록)

이 위에 ClassPilot이 새로 추가하는 기능:
  🆕 다중 역할 (운영자/강사/수강생 분리)
  🆕 수강생 포털 (과제 제출, 복습, 질문)
  🆕 실시간 AI 코파일럿 (수업 중 보조)
  🆕 녹음 → AI 요약
  🆕 이탈 예측 (다중 요인 자동 감지)
  🆕 수강료 관리
  🆕 민원 관리 + AI 초안
```

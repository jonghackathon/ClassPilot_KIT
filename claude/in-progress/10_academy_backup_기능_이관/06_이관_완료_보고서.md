## 5. 기능별 UI 구성 확인

### 커리큘럼 관리 (`/admin/curriculum`)
- PageHero + 클래스 카드 리스트
- 클래스 펼치기/접기 → 단계 → 차시 행
- 차시: 번호 + 제목 + 교재 + 테마 태그
- 프리로드 데이터 5개 클래스 (초4~중1)

### 메모 (`/teacher/memo`)
- 4카테고리 StatusBadge (공지/특이/학생/기타)
- 메모 카드 리스트 + 작성일/수정일
- 아카이브 토글 + 검색
- [+ 메모 작성] 액션

### 월간 보고서 (`/teacher/reports`)
- 학생 선택 + 월 선택
- 보고서 미리보기 (수업 진도 + 출석 + 과제 + AI 성장)
- AI 성장 평가 섹션 (🌱/⭐/📈)
- [인쇄] 버튼

### AI 첨삭 (과제 상세 내 패널)
- 이미지 뷰어 + [AI 첨삭 생성] 버튼
- 4섹션: ⭐ 잘한 점 / ✏️ 교정 / 📝 피드백 / 📈 발전 방향
- [수정] → 텍스트 편집 / [저장]

### 출결 보강
- 숙제 O/X 토글 (emerald/rose)
- 숙제 미완료 시 사유 입력
- 결석/지각/조퇴 시 사유 인라인 입력
- 주별 그리드 뷰 (월~금 × 학생, 클릭 순환)
- 통계 뷰 (학생별 출석률 바)

### 과제 보강
- 3유형 배지 (에세이=indigo, 독서=emerald, 어휘=amber)
- 이미지 업로드 영역 + 미리보기 그리드
- 학생 선택 카드 그리드 + 일괄 생성

### 진도 보강
- 날짜별 수업 내용 기록
- 커리큘럼 차시 선택 연동
- ☑ 과제 자동 생성 토글

### 시간표 학생 배정
- 수업 블록 클릭 → 학생 배정 패널
- 학생 검색 + 체크박스 선택/해제
- 선택된 학생 수 표시

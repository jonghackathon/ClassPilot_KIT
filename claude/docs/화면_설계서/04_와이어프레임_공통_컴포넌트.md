## 2. 데이터 표시 컴포넌트

### 2.1 StatCard — 지표 카드

```
┌──────────────────┐
│  📊 {라벨}        │  ← caption (12px, gray-500)
│                  │
│      {값}        │  ← stat-large (32px, bold)
│    / {부제}       │  ← body (14px, gray-500)
│                  │
│  {트렌드}  ↑ +5  │  ← 12px, green-500 / red-500
└──────────────────┘
```

**변형:**
- `default`: 값 색상 `gray-900`
- `danger`: 값 색상 `red-500` (이탈 위험, 미납 등)
- `warning`: 값 색상 `amber-500`
- `success`: 값 색상 `green-500`

**그리드:** 데스크탑 4열 / 태블릿 2열 / 모바일 2열

### 2.3 DataTable — 데이터 테이블

운영자 화면 전용. 검색·필터·정렬·페이지네이션 내장.

```
┌─────────────────────────────────────────────────────┐
│  {테이블 제목}            [🔍 검색] [필터 ▼] [+ 추가]   │
├─────────────────────────────────────────────────────┤
│  □  │  컬럼1 ↕  │  컬럼2 ↕  │  컬럼3  │  컬럼4  │ 액션 │
├─────┼──────────┼──────────┼────────┼────────┼──────┤
│  □  │  데이터    │  데이터    │  데이터  │ [배지]  │ [···]│
│  □  │  데이터    │  데이터    │  데이터  │ [배지]  │ [···]│
│  □  │  데이터    │  데이터    │  데이터  │ [배지]  │ [···]│
│  □  │  데이터    │  데이터    │  데이터  │ [배지]  │ [···]│
│  □  │  데이터    │  데이터    │  데이터  │ [배지]  │ [···]│
├─────┴──────────┴──────────┴────────┴────────┴──────┤
│  전체 N건                            ← 1 2 3 ... →   │
└─────────────────────────────────────────────────────┘
```

**기능:**
- `↕` 정렬: 컬럼 헤더 클릭 → 오름차순 → 내림차순 → 정렬 해제
- `🔍 검색`: 디바운스 300ms, 엔터 불필요
- `필터 ▼`: 드롭다운 다중 필터 (반, 상태, 위험도 등)
- `□` 체크박스: 다중 선택 (일괄 처리용, Phase 2)
- `[···]` 액션 메뉴: 상세보기, 편집, 삭제 등
- 페이지네이션: 20건/페이지, 번호 + 이전/다음

**모바일 전환:**
```
테이블 → 카드 리스트
┌──────────────────────────┐
│  {이름}  {학년}            │
│  {반 이름}                 │
│  출석률 80%   ● 위험       │
│                   [→]     │
├──────────────────────────┤
│  {이름}  {학년}            │
│  ...                      │
└──────────────────────────┘
```

## 4. 입력 컴포넌트

### 4.1 SearchFilter — 검색 + 필터

```
┌─────────────────────────────────────────────────────┐
│  🔍 이름·이메일 검색        [반 전체 ▼]  [상태 전체 ▼]  │
```

**동작:**

- 필터 드롭다운: 단일 선택, "전체" 옵션 포함
- URL 쿼리 파라미터와 동기화 (새로고침 시 유지)

**모바일:**
```
┌──────────────────────────┐
│  🔍 검색...                │
├──────────────────────────┤
│  [반 전체 ▼]  [상태 ▼]    │
└──────────────────────────┘
```

```
  ┌─────┬─────┬─────┬─────┬─────┬─────┐
```

| 과제 [O]/[X] | 과제 제출 여부 토글. 탭 즉시 API 호출 |
| [전체 출석] | 확인 모달 표시 → 확인 시 반 전체 학생을 "출석"으로 일괄 처리 |

### 인터랙션 설명

| 요소 | 동작 |
|------|------|
| 반 선택 드롭다운 | 담당 반 전환. 선택 시 해당 반의 커리큘럼/진도 로드 |

| 상태 아이콘 | ✅ 완료 = 초록, 🔵 진행중 = 파랑, ⬜ 예정 = 회색 |

| [진도 기록] | 진도 기록 모달 오픈. 단원 선택 + 상태 변경 + 수업일 + 비고 |
| 모달 [저장] | API 호출. 성공 시 테이블 즉시 갱신 + 토스트 |

| API Endpoint | Method | 설명 |
|-------------|--------|------|
| `/api/curriculum?classId={classId}` | GET | 반별 커리큘럼 및 단원 목록 |
| `/api/curriculum/progress?classId={classId}` | GET | 반별 진도 현황 (단원별 상태, 수업일, 비고) |

| `/api/classes?teacherId=me` | GET | 담당 반 목록 (드롭다운용) |

## 공통 사항

### 반응형 브레이크포인트

| 구분 | 너비 | 레이아웃 |
|------|------|---------|
| 모바일 | < 768px | 1열 스택, 버튼 48px 이상, 카드 풀너비 |
| 태블릿 | 768px ~ 1024px | 2열 그리드, 사이드바 접힘 |
| 데스크탑 | > 1024px | 사이드바 고정, 다열 그리드 |

### 에러 처리

| 상황 | 처리 |
|------|------|
| API 요청 실패 | 토스트 에러 메시지 + 재시도 버튼 |
| 낙관적 업데이트 실패 | 자동 롤백 + "저장에 실패했습니다" 토스트 |
| 네트워크 오프라인 | 상단 배너 "오프라인 상태입니다" |
| SSE 연결 끊김 | 자동 재연결 (최대 3회) + "연결 중..." 상태 표시 |

### 접근성

- 모든 버튼 최소 터치 영역 44x44px (모바일 48px)
- 키보드 네비게이션 지원 (Tab, Enter, Escape)
- 색상 외 아이콘/텍스트로 상태 구분 (색각 이상 대응)
- aria-label 필수 적용 (출결 버튼, 상태 뱃지 등)

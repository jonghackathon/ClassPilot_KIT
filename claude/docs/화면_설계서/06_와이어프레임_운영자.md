## 1. 대시보드

**경로:** `/admin/dashboard`

### 와이어프레임

```
+------------------------------------------------------------------+
| AcadeMind                    [알림🔔]  [운영자님 ▼]              |
+------------------------------------------------------------------+
|        |                                                          |
| 사이드 |  📊 대시보드                                             |
| 바     |                                                          |
|        |  +------------+ +------------+ +------------+ +--------+ |
| 대시보드|  | 전체 수강생 | | 오늘 수업  | | 미납 수강료| | 이탈   | |
| 학생   |  |    87명     | |   12반     | |    3명     | | 위험   | |
| 강사   |  | ▲ 5 이번달  | | 진행중 4   | | ₩450,000  | |  2명   | |
| 반관리 |  +------------+ +------------+ +------------+ +--------+ |
| 시간표 |                                                          |
| 수강료 |  +-------------------------------+ +-------------------+ |
| 이탈   |  | 🚨 이탈 위험 학생             | | 📅 오늘 수업 현황 | |
| 예측   |  +-------------------------------+ +-------------------+ |
| 민원   |  | 이름    | 반    | 위험도 | 액션| | 14:00 수학A  3/8 | |
|        |  |---------|-------|--------|-----| | 15:00 영어B  5/6 | |
|        |  | 김민수  | 수학A | ■■■■□  | [→] | | 16:00 수학C  7/10| |
|        |  | 이서연  | 영어B | ■■■□□  | [→] | | 17:00 국어A  4/5 | |
|        |  +-------------------------------+ | 18:00 수학B  6/8 | |
|        |                                    | ...               | |
|        |  +-------------------------------+ +-------------------+ |
|        |  | 📝 미제출 과제 현황           |                       |
|        |  +-------------------------------+                       |
|        |  | 반    | 과제명     | 미제출 | 마감일     |            |
|        |  |-------|------------|--------|------------|            |
|        |  | 수학A | 단원평가3  |  2명   | 2026-04-07 |            |
|        |  | 영어B | Essay #5   |  3명   | 2026-04-08 |            |
|        |  | 국어A | 독후감     |  1명   | 2026-04-09 |            |
|        |  +-------------------------------+                       |
+------------------------------------------------------------------+
```

### 인터랙션 설명

| 요소 | 동작 | 설명 |
|------|------|------|
| KPI 카드 | 클릭 | 해당 상세 페이지로 이동 (수강생→학생목록, 수업→시간표, 미납→수강료, 이탈→이탈예측) |
| 이탈 위험 학생 [→] | 클릭 | `/admin/students/[id]` 학생 상세로 이동 |
| 오늘 수업 행 | 클릭 | `/admin/classes/[id]` 반 상세로 이동 |
| 미제출 과제 행 | 클릭 | 해당 반 상세의 과제 탭으로 이동 |

### 데이터 연결

| API | 메서드 | 용도 |
|-----|--------|------|
| `/api/lessons` | GET | 오늘 수업 현황, 수업 수 KPI |
| `/api/ai/churn` | GET | 이탈 위험 학생 리스트, 이탈 위험 KPI |
| `/api/payments` | GET | 미납 수강료 KPI |
| `/api/students` | GET | 전체 수강생 수 KPI |
| `/api/assignments` | GET | 미제출 과제 현황 |

- **자동 새로고침:** SWR `refreshInterval: 300000` (5분)
- **캐시 전략:** `stale-while-revalidate`, 페이지 포커스 시 즉시 갱신

### 모바일 변환

- KPI 카드: 2x2 그리드
- 하단 2컬럼 → 1컬럼 세로 스택 (이탈위험 → 오늘수업 → 미제출과제 순서)
- 사이드바 → 하단 탭 네비게이션

### 학생 등록 모달

```
+--------------------------------------------+
| 학생 등록                            [✕]   |
|--------------------------------------------|
| 이름 *      [                           ]  |
| 이메일 *    [                           ]  |
| 학교        [                           ]  |
| 학년 *      [중1 ▼]                        |
| 수강반 *    [□ 수학A  □ 영어B  □ 국어A]    |
|                                            |
| ── 학부모 정보 ──                          |
| 학부모 이름 [                           ]  |
| 연락처      [                           ]  |
| 이메일      [                           ]  |
|                                            |
|            [취소]  [등록하기]               |
+--------------------------------------------+
```

### 인터랙션 설명

| 요소 | 동작 | 설명 |
|------|------|------|
| 검색 입력 | 입력 (debounce 300ms) | 이름 또는 이메일로 실시간 필터링 |
| 필터 드롭다운 | 선택 | 반/상태(재원·휴원·퇴원)/위험도(높음·주의·정상)별 필터 |
| 정렬 버튼 | 클릭 | 이름(가나다)/출석률(%)/위험도(높→낮) 기준 정렬 토글 |
| [→] 액션 | 클릭 | `/admin/students/[id]` 학생 상세로 이동 |
| [+ 학생 등록] | 클릭 | 등록 모달 오픈 |
| 등록 모달 [등록하기] | 클릭 | 유효성 검증 후 POST, 성공 시 목록 갱신 + 토스트 |
| 페이지네이션 | 클릭 | 20건 단위 페이지 전환 |

### 데이터 연결

| API | 메서드 | 용도 |
|-----|--------|------|
| `/api/students` | GET | 학생 목록 조회 (`?page=&limit=20&search=&class=&status=&risk=&sort=`) |
| `/api/students` | POST | 학생 등록 |
| `/api/classes` | GET | 필터 드롭다운용 반 목록 |
| `/api/ai/churn` | GET | 이탈 위험도 데이터 |

### 인터랙션 설명

| 요소 | 동작 | 설명 |
|------|------|------|
| ← 학생 목록 | 클릭 | `/admin/students`로 복귀 |
| 탭 (출결/과제/상담/결제) | 클릭 | 탭 전환, URL 쿼리 파라미터로 상태 유지 (`?tab=attendance`) |
| 달력 날짜 | 클릭 | 해당 일자 출결 상세 팝오버 (수업명, 시간, 출결상태) |
| 달력 ◀▶ | 클릭 | 월 전환 |
| [+ 상담 기록 추가] | 클릭 | 상담 기록 모달 오픈 |
| 이탈위험도 바 | 호버 | 세부 요인 툴팁 표시 |
| [수정] | 클릭 | 기본 정보 인라인 편집 모드 전환 |
| [삭제] | 클릭 | 확인 다이얼로그 후 소프트 삭제 |

### 데이터 연결

| API | 메서드 | 용도 |
|-----|--------|------|
| `/api/students/[id]` | GET | 학생 기본 정보 |
| `/api/students/[id]` | PUT | 학생 정보 수정 |
| `/api/students/[id]` | DELETE | 학생 삭제 (소프트) |
| `/api/students/[id]/attendance` | GET | 출결 달력 데이터 |
| `/api/students/[id]/assignments` | GET | 과제 제출 현황 |
| `/api/students/[id]/consultations` | GET | 상담 타임라인 |
| `/api/students/[id]/consultations` | POST | 상담 기록 추가 |
| `/api/students/[id]/payments` | GET | 결제 납부 이력 |
| `/api/ai/churn/[studentId]` | GET | 개별 이탈 위험도 |

### 모바일 변환

- 좌측 기본정보 → 상단 요약 카드 (접이식)
- 우측 탭 → 하단 전체 너비 탭
- 달력 → 세로 스크롤 리스트 뷰 옵션 제공
- 기본정보 요약: 이름 + 학년 + 이탈위험 바만 표시, [더보기]로 전체 펼침

### 인터랙션 설명

| 요소 | 동작 | 설명 |
|------|------|------|
| 검색 입력 | 입력 (debounce 300ms) | 이름/이메일 실시간 필터링 |

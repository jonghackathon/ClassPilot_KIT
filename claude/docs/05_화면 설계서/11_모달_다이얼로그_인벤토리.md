## 1. 모달 공통 규격

### 1.1 크기

| 크기 | 너비 | 용도 |
|------|------|------|
| **sm** | 400px | 확인 다이얼로그 |
| **md** | 560px | 폼 모달 (등록/수정) |
| **lg** | 720px | 뷰어 모달 (이력 조회 등) |

### 1.2 모바일 변환

| 데스크탑 | 모바일 (~768px) |
|---------|----------------|
| sm 모달 | 센터 모달 (전체 너비 - 32px 마진) |
| md 모달 | 전체 화면 바텀시트 (하단에서 슬라이드 업) |
| lg 모달 | 전체 화면 |

### 1.3 공통 동작

| 동작 | 설명 |
|------|------|
| 열기 | 오버레이 fade-in + 모달 scale-in (또는 바텀시트 slide-up) |
| 닫기 | ESC 키 / 오버레이 클릭 / [×] 버튼 / [취소] 버튼 |
| 변경 보호 | 폼에 변경사항이 있으면 닫기 시 확인 다이얼로그 표시 |
| 포커스 트랩 | 모달 내부에서만 Tab 순환 |
| 스크롤 잠금 | 모달 열린 동안 body 스크롤 비활성 |
| z-index | 오버레이: 40 / 모달: 50 / 중첩 모달: 60 |

## 5. 모달 접근성 체크리스트

| 항목 | 적용 |
|------|------|
| 포커스 트랩 | 모달 내부에서만 Tab 순환 |
| 열릴 때 포커스 | 첫 번째 인터랙티브 요소 (또는 닫기 버튼) |
| 닫힐 때 포커스 | 모달을 연 트리거 버튼으로 복귀 |
| ESC 키 | 모달 닫기 |
| role | `role="dialog"`, `aria-modal="true"` |
| aria-label | 모달 제목과 동일 |
| body 스크롤 | 모달 열린 동안 잠금 |
| 오버레이 | `aria-hidden="true"` |

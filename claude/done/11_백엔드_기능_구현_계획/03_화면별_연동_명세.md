# 화면별 백엔드 연동 명세

> 현재 `src/app` 기준으로 운영 중인 화면들이 필요로 하는 API, 상태 전환(useState -> SWR), 모달/폼 엔드포인트를 정리한 문서.
> 페이지 수는 프론트 구현 진행에 따라 계속 바뀌므로, 고정 카운트 대신 실제 라우트 구조 기준으로 유지한다.

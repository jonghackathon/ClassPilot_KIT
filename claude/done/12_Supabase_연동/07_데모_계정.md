## 시드 데이터 재실행

```bash
node prisma/seed.js
```

- `npx prisma db seed` 사용 불가 (prisma.config.ts에 seed 설정 없음)
- 시드 실행 시 기존 데이터 전체 삭제 후 재삽입됨 (resetDatabase 함수)

**Why:** 시드 파일에 `resetDatabase()` 호출이 먼저 실행됨. 재실행 시 운영 DB에 절대 실행 금지.
**How to apply:** 로컬/개발 DB에서만 실행. 계정 데이터 날아가도 되는 환경에서만 사용.

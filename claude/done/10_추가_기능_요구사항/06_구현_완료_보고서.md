## 3. DB 변경 사항

### 신규 모델
- `Curriculum`, `Stage`, `Lesson` — 커리큘럼 계층 구조
- `MonthlyReport` — 월간 보고서 + AI 성장 평가
- `Memo` — 메모/노트
- `AppSetting` — 학원 설정 key-value

### 기존 모델 컬럼 추가
- `Attendance`: `homeworkDone`, `homeworkNote`, `absenceReason`
- `Assignment`: `type` (ESSAY/READING/VOCAB), `imageUrl`
- `LessonLog`: `content`, `curriculumLessonId`
- `Student`: `memo`, `scheduleId`

---

## 4. 구현 방식

- **커리큘럼·메모·설정·고도화 기능** — Codex 에이전트 (패턴 명확, 범위 독립)
- **월간 보고서 AI 성장 평가** — Claude 직접 처리 (SSE + Claude API 설계)
- **AI 에세이 첨삭** — Claude 직접 처리 (멀티모달 + 구조화 출력 설계)

---

## 5. 특이 사항

- 커리큘럼 모델은 JSONB 대신 정규화된 3개 모델로 구현 — 필터·정렬 유연성 확보
- 에세이 첨삭 이미지는 Supabase Storage 업로드 후 signed URL로 Claude에 전달
- 월간 보고서 인쇄는 `@media print` CSS로 별도 인쇄 레이아웃 적용

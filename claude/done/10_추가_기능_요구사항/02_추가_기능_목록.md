# 추가 기능 목록

**작성일:** 2026.04.09  
**기준:** 강사 인터뷰 + 워크플로우 분석 결과

## 1. 신규 기능 (완전히 새로 개발)

### A. 커리큘럼 관리 시스템

**왜 필요:** 과제 생성, 수업 진도, 월간 보고서의 기반 데이터. 이것 없으면 나머지 3개 기능이 구조 없이 동작.

| 항목 | 상세 |
|------|------|
| 계층 구조 | 클래스 → 단계(Stage) → 차시(Lesson) |
| 차시 속성 | 번호(901-1), 제목, 교재, 테마/기법 태그 |
| 기본 데이터 | 초4~중1 독서논술 커리큘럼 기본 데이터 |
| CRUD | 클래스/단계/차시 추가/수정/삭제 |
| UI | 펼치기/접기 트리, 차시 수 카운트 |
| 연동 대상 | 과제 제목 자동완성, 수업 진도 차시 선택 |
| 적용 페이지 | `/admin/curriculum` |
| 우선순위 | **높음** — 다른 기능의 기반 |

### B. 월간 보고서 + AI 성장 평가

**왜 필요:** 학부모 커뮤니케이션의 핵심. 학원 재원율에 직접 영향.

| 항목 | 상세 |
|------|------|
| 보고서 구성 | 학원 로고 + 학생 정보 + 수업 진도 + 출석 현황 + 과제 목록 + AI 성장 평가 |
| AI 성장 평가 | Claude API → 🌱이달의 성장 / ⭐잘하고 있어요 / 📈앞으로의 방향 |
| 인쇄 | `@media print` 최적화, 페이지 나눔 방지 |
| 저장 | 코멘트 + AI 성장 텍스트 DB 저장 |
| 적용 페이지 | `/teacher/reports` |
| 우선순위 | **높음** |

### C. AI 에세이 첨삭

**왜 필요:** 강사의 시간 절약 + 학생 피드백 품질 향상. 매일 사용하는 핵심 기능.

| 항목 | 상세 |
|------|------|
| 이미지 첨삭 | 학생 손글씨 사진 → Claude 멀티모달 → 구조화된 피드백 |
| 텍스트 첨삭 | 학생 타이핑 글 → Claude 텍스트 → 동일 구조 피드백 |
| 피드백 구조 | ⭐잘한점 / ✏️교정 / 📝구성피드백 / 📈발전방향 |
| 수동 편집 | AI 결과를 강사가 수정 후 저장 |
| 적용 페이지 | `/teacher/assignments/[id]` 내 패널 |
| 우선순위 | **높음** |

### D. 메모/노트 시스템

**왜 필요:** 강사가 수시로 기록하는 학생 특이사항, 학부모 통화 내용 등을 한 곳에.

| 항목 | 상세 |
|------|------|
| 카테고리 | 공지사항(파랑) / 특이사항(주황) / 학생 메모(초록) / 기타(보라) |
| 아카이브 | 1주 경과 시 자동 아카이브, 수동 아카이브/복원 |
| 검색 | 제목 + 내용 통합 검색 |
| UI | 카드형 리스트, 카테고리 필터, 날짜 구분선 |
| 적용 페이지 | `/teacher/memo` |
| 우선순위 | **중~높음** |

### E. 설정 페이지

**왜 필요:** AI 기능 API 키 설정, 보고서 인쇄용 학원 로고 관리.

| 항목 | 상세 |
|------|------|
| 기능 | 로고 업로드, Claude API 키 설정, 비밀번호 변경 |
| 저장 | key-value 테이블 (AppSetting) |
| 적용 페이지 | `/admin/settings` |
| 우선순위 | **중간** |

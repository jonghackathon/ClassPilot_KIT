# 기능 정의서

**작성일:** 2026.04.10  
**버전:** v2.0 (추가 기능 반영)  
**기준:** Claude 설계 검토 완료 후 확정된 기능 정의

---

## 1. 한 줄 정의

> ClassPilot은 독서논술 학원을 위한 AI 기반 학원 관리 SaaS로,  
> 강사의 실제 수업 워크플로우(출결→수업→과제→첨삭→보고서)를 한 플랫폼에서 지원한다.

---

## 2. 역할별 핵심 기능

### 운영자 (ADMIN)

| 기능 | 설명 |
|------|------|
| 커리큘럼 관리 | 클래스→단계→차시 계층 구조 CRUD |
| 학원 설정 | 로고, API 키, 운영 정보 |
| 학생/강사 관리 | 등록, 역할 배정, 시간표 연동 |
| 수강료 관리 | 납부 추적, 미납 알림 |
| 이탈 예측 | AI 기반 위험도 분석 |
| 민원 관리 | 학부모 민원 + AI 초안 |

### 강사 (TEACHER)

| 기능 | 설명 |
|------|------|
| 출결 관리 | 출결 입력 + 숙제 O/X + 결석 사유 인라인 |
| 과제 관리 | 유형별 생성, 이미지 업로드, AI 에세이 첨삭 |
| 수업 진도 | 날짜별 수업 내용 기록, 과제 자동 생성 |
| 월간 보고서 | AI 성장 평가 + 인쇄 최적화 |
| AI 코파일럿 | 수업 중 실시간 질문 답변 (SSE) |
| 녹음 요약 | Whisper + Claude 수업 요약 |
| 메모 | 4카테고리, 자동 아카이브, 검색 |

### 수강생 (STUDENT)

| 기능 | 설명 |
|------|------|
| 홈 | 출결 현황, 최근 과제, 공지 |
| 출결 조회 | 내 출석 기록 |
| 과제 조회 | 과제 목록 + 첨삭 피드백 확인 |
| AI 복습 | quiz + 복습 자료 |
| Q&A 봇 | 질문 → AI 1차 → 강사 전달 |

## 3. 기술 스택

| 항목 | 기술 |
|------|------|
| Frontend | Next.js 16, React, Tailwind CSS |
| Backend | Next.js API Routes, Prisma ORM |
| DB | Supabase PostgreSQL |
| AI | Claude API (멀티모달, SSE), OpenAI Whisper |
| Auth | NextAuth v5, bcrypt |
| 배포 | Vercel |

---

## 4. 주요 UX 원칙

- **출결이 기본 화면** — 강사는 앱을 열면 출결부터 시작
- **숙제+출결 통합** — 별도 페이지 없이 한 화면에서 처리
- **결석 사유 인라인** — 결석 선택 즉시 사유 입력란 표시
- **과제 자동 생성** — 수업 진도 기록 체크 한 번으로 반 전체 과제 생성
- **AI 첨삭 즉시** — 사진 업로드 → AI 피드백 → 수정 → 저장
- **인쇄 최적화** — 보고서는 `@media print`로 깔끔하게

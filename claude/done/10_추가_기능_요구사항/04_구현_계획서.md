# 구현 계획서

**작성일:** 2026.04.10  
**목표:** 추가 요구사항 10개 기능을 ClassPilot에 구현  
**원칙:** ClassPilot 디자인 유지, Step 별 순차 구현

---

## 1. 구현 순서 및 의존관계

```
커리큘럼 관리 (기반)
    ↓
출결 고도화 ──┐
과제 고도화 ──┤ 병렬 가능 (write scope 분리)
수업 진도 고도화 ─┘
    ↓
메모 시스템 (독립)
설정 페이지 (독립)
    ↓
월간 보고서 (커리큘럼 + 출결 + 과제 데이터 집계)
AI 에세이 첨삭 (과제 데이터 연동)
```

**핵심:** 커리큘럼이 먼저 완료되어야 과제 자동완성, 수업 진도 차시 선택, 월간 보고서가 온전히 동작한다.

---

## 2. 단계별 구현 계획

### Phase 1 — 기반 인프라 (순차)

```
브랜치: codex/step8-curriculum-foundation
  → prisma/schema.prisma 확장 (신규 모델 추가)
  → DB 마이그레이션
  → /api/curriculum/** 구현
  → /admin/curriculum 페이지 구현
```

### Phase 2 — 핵심 기능 (병렬 가능)

```
codex/step8-attendance-enhance   → 출결 고도화
codex/step8-assignment-enhance   → 과제 고도화
codex/step8-progress-enhance     → 수업 진도 고도화
```

### Phase 3 — 독립 기능

```
codex/step8-memo           → 메모 시스템
codex/step8-settings       → 설정 페이지
```

### Phase 4 — AI 연동 기능 (Claude 직접 처리)

```
claude: step8-reports-ai   → 월간 보고서 + AI 성장 평가
claude: step8-essay-ai     → AI 에세이 첨삭 (멀티모달)
```

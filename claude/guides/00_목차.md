## 빠른 참조

```bash
# Claude Code 실행
claude                    # 대화형 모드
claude "질문"             # 원샷 모드
claude -p "질문"          # 파이프/스크립트 모드 (비대화형)

# 주요 슬래시 커맨드
/help                     # 도움말
/compact                  # 컨텍스트 압축
/clear                    # 대화 초기화
/commit                   # Git 커밋
/review-pr                # PR 리뷰
/init                     # CLAUDE.md 생성
```

## 환경별 사용 방법

| 환경 | 설명 |
|------|------|
| **CLI** | 터미널에서 `claude` 명령어로 실행 |
| **VS Code** | 확장 프로그램 설치 → 사이드바 Claude 패널 |
| **JetBrains** | 플러그인 설치 → 도구 창 Claude 패널 |
| **Desktop App** | Mac/Windows 전용 네이티브 앱 |
| **Web App** | claude.ai/code 에서 브라우저로 사용 |
